🌐 什么是 Web 开发？

Web 开发是指构建网站或 Web 应用程序的过程。从简单的静态网页到复杂的动态应用，Web 开发涵盖了广泛的技能和技术。

📐 Web 开发的三大基石

1. HTML — 结构

HTML（超文本标记语言）是网页的骨架。它定义了网页的内容和结构。

2. CSS — 样式

CSS（层叠样式表）负责网页的外观。它让网页变得美观。

3. JavaScript — 交互

JavaScript 赋予网页动态交互能力，让用户可以与页面进行互动。

🛤️ 学习路径建议

阶段 内容 时间
第一阶段 HTML + CSS 基础 2-3周
第二阶段 JavaScript 核心 3-4周
第三阶段 前端框架 (Vue/React) 4-6周
第四阶段 后端基础 (Node.js) 3-4周
第五阶段 项目实战 持续进行

📚 推荐资源

  • MDN Web Docs — 最权威的 Web 技术文档
  • freeCodeCamp — 免费的交互式编程学习平台
