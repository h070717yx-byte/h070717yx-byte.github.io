🤔 什么是 Git？

Git 是目前世界上最流行的分布式版本控制系统。它能追踪文件的每一次修改，让你轻松地进行团队协作和代码管理。

📋 为什么需要版本控制？

  • ✅ 记录历史 — 每次修改都有记录，随时可以回溯
  • ✅ 多人协作 — 多人同时修改代码而不会互相覆盖
  • ✅ 分支管理 — 在不同功能之间自由切换，互不干扰
  • ✅ 备份代码 — 代码托管在远端，不怕丢失

💡 最佳实践

  1. 经常提交 — 小步快跑，每次提交只做一件事
  2. 写好注释 — commit message 要清晰明了
  3. 合理分支 — 主分支保持稳定，开发在特性分支进行
  4. 及时同步 — 开始工作前先 pull，完成后及时 push
